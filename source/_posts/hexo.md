### 写博客常用命令
- [hexo Docs](https://hexo.io/docs/writing)
 - 博客本地启动:  hexo s  = hexo server
 - 本地创建一个新的页面: hexo new [title]
 - 写完之后生成静态文件: hexo g = hexo generate
 - 提交到远程git上: hexo d   = hero deploy
